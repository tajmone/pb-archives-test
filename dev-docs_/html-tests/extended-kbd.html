Within Paragraphs

To copy selection in clipboard use Ctrl+C; and then to paste it use Ctrl+V. You can always undo with Ctrl+Z!

Longer keystrokes: Alt+Shift+F1

Let’s test some colors with <mark> — !mark: Ctrl+Shift+F12; !markRed: Alt+Shift+F4; !markGreen: Ctrl+Alt+Tab; !markBlue: Ctrl+Alt+→;

Keystrokes styles:

  • normal: Ctrl+Z
  • bold:: Alt+Shift+F1
  • italic: Alt+Shift+F1
  • strikeout: Alt+Shift+F1

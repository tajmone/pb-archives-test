Alerts Test

Alerts PP-Macros Test

This is a run test of the Alerts pp-macros set.

Files List

  • macros files:
    • Alerts.pp — macros definition module.
    • Alerts.css — CSS style for Alerts elements.
  • extra files:
    • github.css — basic document CSS styling.

Macros list

  • !Alert(CONTENTS) — Default alert (blue)
  • !AlertNote(CONTENTS) — Plain alert (grey)
  • !AlertError(CONTENTS) — Error alert (red)
  • !AlertWarn(CONTENTS) — Warning alert (yellow)
  • !AlertSuccess(CONTENTS) — Success alert (green)

Default Alert

macro used:

!Alert(__Default__ — Lorem ipsum dolor sit amet, pri cu libris dicunt.)

raw html output:

<div class="alert-messages">
<div class="alert ">
__Default__ — Lorem ipsum dolor sit amet, pri cu libris dicunt.
</div>
</div>

result preview:

Default — Lorem ipsum dolor sit amet, pri cu libris dicunt.

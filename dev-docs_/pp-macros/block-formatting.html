QuoteSource Macro

The !QuoteSource macro is used to render the source of a blockquote.

MARKDOWN SOURCE:

> A block quote with some text. Blah, blah, blah.


> 


> <p class="source">— the source of the text</p>

HTML OUTPUT:

<blockquote>


<p>A block quote with some text. Blah, blah, blah.</p>


<p class="source">


— the source of the text


</p>


</blockquote>

VISUAL RESULT:

A block quote with some text. Blah, blah, blah.

— the source of the text

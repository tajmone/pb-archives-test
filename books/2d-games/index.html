Programming 2D Scrolling Games

This folder contains the PDF book Programming 2D Scrolling Games (J. Logsdon and D. Siqueira, 2014) and the source code files that came with it. This used to be a commercial book, from 2005 to 2014, then the author released it under Creative Commons.

This is the 2014/02/06 edition, updated for PureBasic 4.61 & 5.0 — as posted by the author on PureBASIC forums.

Files

  • Programming_2D_Scrolling_Games.pdf — the book in PDF, as released by the author in 2014/02/06. Some PDF metadata added.
  • pb_book_source.zip — the book source examples, as released by the author in 2014/02/06.
